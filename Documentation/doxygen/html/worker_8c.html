Detailed Description

Worker process implementation for RT-THM.

Each worker simulates tasks, updates shared memory stats, and responds to pause/resume commands from the supervisor.

void worker_process ( int  id)

Main entry point for worker process.

Parameters
idWorker identifier (0 to MAX_WORKERS-1)

Infinite loop that:

  • Checks for pause/resume commands from parent via shared memory
  • Simulates work with random sleep (1-3s normal, 5s paused)
  • Updates shared memory with health stats atomically
  • Simulates random crash (5% probability) for testing auto-restart
    Note
    This function never returns normally; calls exit() on crash
    Warning
    Must be called only in child process after fork()
