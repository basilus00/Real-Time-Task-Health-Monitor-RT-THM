Manages cleanup, worker pause/resume signals, and graceful shutdown.

\begin{DoxyParams}{Parameters}
{\em sig} & Signal that triggered cleanup (SIGINT/\+SIGTERM)\\
\hline

Performs in order\+:
\begin{DoxyEnumerate}
\item Restores terminal via \doxylink{config_8c_a4aea176d6dafe443d89b8d850830709a}{endwin()}
\item Sends SIGTERM to all workers
\item Detaches and removes shared memory
\item Removes semaphore
\item Closes log file
\item Exits via \doxylink{config_8c_a4aea176d6dafe443d89b8d850830709a}{\+\_\+exit()} (signal-\/safe) \begin{DoxyNote}{Note}
Uses cleanup\+\_\+done flag to prevent double-\/execution 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
