\doxysubsection{Detailed Description}
Worker process implementation for RT-\/\+THM. 

Each worker simulates tasks, updates shared memory stats, and responds to pause/resume commands from the supervisor.

Infinite loop that\+:
\begin{DoxyItemize}
\item Checks for pause/resume commands from parent via shared memory
\item Simulates work with random sleep (1-\/3s normal, 5s paused)
\item Updates shared memory with health stats atomically
\item Simulates random crash (5\% probability) for testing auto-\/restart \begin{DoxyNote}{Note}
