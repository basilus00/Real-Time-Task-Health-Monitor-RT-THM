\doxysubsection{Detailed Description}
Ncurses dashboard implementation for RT-\/\+THM. 

Provides real-\/time visual interface with system info, worker table, and live logs. Uses double-\/buffering and atomic snapshots for flicker-\/free, thread-\/safe rendering.
