\doxysubsection{Detailed Description}
RT-\/\+THM supervisor process (parent) implementation. 

Main entry point\+: initializes IPC, spawns workers, runs ncurses dashboard, and monitors worker health with auto-\/restart on failure.

Main entry point for RT-\/\+THM supervisor. 

\begin{DoxyReturn}{Returns}
EXIT\+\_\+\+SUCCESS on normal exit, EXIT\+\_\+\+FAILURE on initialization error
\end{DoxyReturn}
Execution flow\+:
\begin{DoxyEnumerate}
\item Initialize logger and load config.\+txt
\item Create shared memory and semaphore for IPC
\item Fork worker processes
\item Initialize ncurses UI dashboard
\item Enter supervision loop\+:
\begin{DoxyItemize}
\item Handle user input (q=quit, r=restart, resize)
\item Draw dashboard with worker stats
\item Monitor workers via \doxylink{config_8c_a4aea176d6dafe443d89b8d850830709a}{waitpid()} and restart on crash
\item Check timeouts and send signals as needed
\end{DoxyItemize}
\item Cleanup resources on exit \begin{DoxyNote}{Note}
This function contains the main infinite loop; normal exit via \textquotesingle{}q\textquotesingle{} key or SIGINT 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
